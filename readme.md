## Project Overview
This project is an AI-powered health monitoring system that detects early anomalies in physiological data such as heart rate, blood oxygen levels, sleep patterns, and activity levels. The system uses machine learning (Isolation Forest) to provide real-time anomaly detection, alerts, and preventive recommendations.

**SDG Focus:** SDG 3 â€“ Good Health & Well-Being  
This system contributes to ensuring healthy lives and promoting well-being through early detection of potential health risks.

---

## Features
- Real-time anomaly detection in health data  
- Visual dashboard with interactive charts  
- Alerts and preventive recommendations for anomalies  
- Downloadable anomaly reports  
- Simulated health data generation for testing

## Installation & Setup
1. Clone this repository:  
```bash
git clone https://github.com/UmChristelle/health-anomaly-detection.git
cd health-anomaly-detection
```
2. Install dependencies:
   
   pip install -r app/requirements.txt

   
3. Run the Streamlit dashboard locally:

   streamlit run app/app.py

   
4. pen your browser at the address provided by Streamlit (usually http://localhost:8501).
   Live Demo

Access the deployed dashboard here:
https://health-anomaly-detection.onrender.com
